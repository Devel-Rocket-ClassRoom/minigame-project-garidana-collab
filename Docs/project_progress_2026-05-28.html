게임 개요

김경일의 모험: Origin은 3D 뷰 액션 RPG입니다. 플레이어는 8방향 이동과 3단 콤보 공격, 대시를 사용해 몬스터를 처치하고, 경험치와 골드를 획득해 레벨업하며 공격을 강화합니다.

핵심 플레이 루프는 퀘스트 수락 → 몬스터 처치 / 아이템 수집 → 퀘스트 완료 → 레벨업 / 공격 강화 → 다음 구역 진입으로 설계되어 있습니다.

Low-poly 아트 스타일 기반의 여러 구역(시작 마을, 어둠의 숲, 황무지/사막 구역)으로 구성되며, NPC 및 상자 상호작용, 인벤토리(장비창) 시스템을 2주차에서 완성 목표로 진행 중입니다.
