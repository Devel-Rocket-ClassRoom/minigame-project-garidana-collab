### 빌드 노트

게임 : 김경일의 모험 : Origin
담당자 : 오준환
제작자 : 나상욱


### 2주차 빌드 추가 구현 현황

# UI 구현
    - 플레이어 사망시 GameOverUi 팝업
    - 5초 타이머 후 Respawn 버튼 클릭 가능
# 상호작용 시스템 구현 
    - IInteractable
# 플레이어 HUD 개선 
    - Radial360으로 사용시 대시 및 회복 쿨다운 시각 추가
    - Floating Text
        - 몬스터 타격시 들어간 데미지 
        - 플레이어 피격시 들어온 데미지 
        - 몬스터 처치후 경험치 획득시 몬스터 시체위 경험치 
        - 골드 사용 혹 획득 골드 
# 전체 맵 디자인 완료
    - 시작 구역 (천호마을) : 완료
    - 황무지 구역 : 완료
    - 어둠 숲 구역 : 콜라이더 X
    - 오크 주둔지 구역 : 콜라이더 X
    - 왕의 무덤 구역 : 계단 및 경사로 
# 타격감 개선
    - 몬스터 피격시 넉백 추가
    - 플레이어 공격 유효시 히트 스탑 추가 (히트 스탑 시간 조절가능)
# 퀘스트 시스템 구현 완료
    - 퀘스트 진행 가능 - NPC 상호작용 가능
    - 퀘스트 진행 중 대화 불가
    - 퀘스트 완료가능 - NPC 상호작용 가능
    - 아이템 수집 퀘스트의 경우 퀘스트 진행중에만 해당 퀘스트 아이템 드랍
    - 퀘스트 완료시 레벨업 or 경험치 획득 및 골드 획득
# 플레이어 공격 강화 시스템 구현 완료
    - 10 / 20 / 30 / 40 레벨 달성시 공격 데미지 배율 및 범위 증가, 이펙트 변경
# 황무지 몬스터 구현 완료
    - Bumble
    - Sting
    - Shade
    - Toadstool
# 어둠 숲 몬스터 구현 완료
    - Shadow
    - Phantom
    - Spider King
    - Queen Worm
    - Nightmare
    - Death
    - Death Mage (엘리트)
# 웨이포인트 일부 구현
    - 천호마을과 황무지 구역 웨이포인트 활성화 시 상호 순간이동가능.
    - 리스폰 시 가장 최근에 활성화한 웨이포인트 위치에서 부활.
        - 활성화한 웨이포인트 없을시 게임 시작 스폰포인트에서 부활.
