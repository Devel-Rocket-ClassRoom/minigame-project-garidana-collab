### 빌드 3 계획
1. 플레이어 장비창 구현
    - 플레이어 현재 스탯 ( 레벨, 공격력, 공격 강화 단계, 체력, 완료 챕터 (퀫스트))
    - 현재 착용중인 장비 표시 섹션
    - 현재 보유중인 장비 인벤토리 섹션
2. 웨이포인트 시스템 완성
    - 천호마을 제외 모든 타 구역 (현재 시스템 상 Region 구역)들의 웨이포인트는 천호마을로 이동
    - 천호마을 웨이포인트에서는 타구역 선택 후 이동
3. 가이드 / 튜토리얼 시스템
    - 플레이어가 플레이 후 처음 회복 (기본제공 2개) 를 소진 했을시 팝업으로 회복 갯수 충전 방법 가이드
    - 플레이어가 진행 가능한 퀘스트 주변으로 가면 방향을 알려주는 가이드 시스템 구현
4. 보스 구현 
    - 보스 간단한 패턴 및 몬스터 소환 메케니즘 구현
5. 세이브 로드 시스템 구현
    - 웨이포인트 상호작용 시 세이브 진행 (이동 시)
    - 세이브 내용
        - 레벨
        - 경험치
        - 공격 강화 단계
        - 해금 웨이포인트
        - 퀘스트 현황 
        - 장비 현황
        - 총 플레이 시간
        - 처치한 몬스터 수
6. 미니맵 구현 
    - 좌측 상단 PlayerHud에 현재 위치와 주변 지형을 보여주는 미니맵 구현
7. 사운드 작업
    - 사운드트랙
    - 효과음 
        - 버튼 클릭
        - 플레이어 기합
        - 플레이어 피격 신음
        - 플레이어 사망
        - 몬스터 피격 
        - 대쉬 
        - 레벨업
        - 웨이포인트 이동
