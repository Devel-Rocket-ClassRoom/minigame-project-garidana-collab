## 핵심 메커닉 요약

### 게임 루프
```
퀘스트 수락 
  → 몬스터 처치 
    → 아이템/경험치 획득 
      → 퀘스트 완료 
        → 능력 강화 
          → 다음 구역 진입
```

### 플레이 조작
- **이동:** 방향키
- **대시:** Shift (스태미나 소모)
- **공격:** Space (3단 콤보)
- **상호작용:** F (퀘스트 수락/완료, 오브젝트)
- **메뉴:** Esc (일시 정지)

### 성장 시스템
- 몬스터 처치 → 경험치 획득
- 경험치 누적 → 레벨업
- 레벨 10/20/30 도달 → 공격 강화

## 기술 스택 및 구조

### 핵심 시스템
- **전투:** PlayerAttack, BaseMonster, IDamageable
- **상호작용:** IInteractable, PlayerInteractor
- **퀘스트:** 퀘스트 데이터 구조, 상태 관리
- **강화:** PlayerAttackUpgrade
- **회복:** PlayerHealing
- **효과:** HitStopManager, 넉백 로직
- **HUD:** PlayerHud, UI 리플렉션

### 구조적 특징
- **인터페이스 기반 설계:** IDamageable, IInteractable로 확장성 확보
- **스크립터블 오브젝트:** 퀘스트, 아이템, 몬스터 데이터 분리
- **파라미터화:** 대부분의 수치를 에디터에서 조정 가능
- **재사용 가능:** BaseMonster 상속으로 새 몬스터 추가 용이
